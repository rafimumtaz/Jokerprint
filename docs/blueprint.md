# **App Name**: Joker Catalog

## Core Features:

- Product Creation: Allow staff to add new products with details like name, description, price, and images.
- Product Editing: Enable modification of existing product information, including details and images.
- Categorization with Tags: Implement a tagging system for organizing products into categories.
- Search Functionality: Enable users to quickly find products using keywords. The tool can search product names and descriptions and make sure the returned information aligns with products.

## Style Guidelines:

- Primary color: Light pastel green (#C1FFC1) to create a soothing and inviting atmosphere.
- Background color: Off-white (#FAF9F6) to provide a clean and neutral backdrop that complements the pastel green.
- Accent color: Muted yellow (#FAF884), subtly different from the primary color, will indicate interactive elements without distracting from other functions.
- Font pairing: 'Inter' for body text (sans-serif) paired with 'Literata' for headlines (serif) to balance modern clarity with a touch of traditional elegance.
- Use clean, simple icons to represent product categories and actions, ensuring clarity and ease of use.
- Employ a clean, card-based layout to showcase products, making navigation intuitive and visually appealing.
- Incorporate subtle hover effects and transitions to enhance interactivity without overwhelming the user.
